# KALI — Kinetic Agentic Learning Intelligence
## Dedicated Quantum Computing Academy (India)

KALI is a specialized AI-driven educational portal designed to democratize quantum computing concepts for Indian students. By combining interactive curricula, real-world financial optimization, and a friendly AI mentor, KALI makes the complex world of qubits, gates, and algorithms approachable for everyone.

### 🌟 Key Features
- **Quantum Education Core**: 5 interactive lessons (Qubits to Algorithms) with mastery-based quizzes.
- **BEE Portfolio Optimizer**: Apply Quantum Genetic Algorithms (QGA) to 15 blue-chip NSE stocks (Reliance, TCS, etc.) and compare against the NIFTY 50.
- **Stable AI Brain**: KALI, your dedicated quantum mentor, powered by Groq (Llama3) with a strict educator persona and simple English focus.
- **India-Centric Data**: All stock data, benchmarks, and research examples are sourced from the Indian ecosystem (NSE, ISRO, IITs).

### 🚀 Getting Started
#### 1. Live Deployment
Visit the KALI Academy: [https://kali-quantum-academy.streamlit.app](https://kali-quantum-academy.streamlit.app)

#### 2. Run Locally
```bash
# Clone the repository
git clone https://github.com/yourusername/kali-project

# Install dependencies
pip install -r requirements.txt

# Configure Secrets (Create .streamlit/secrets.toml)
GROQ_API_KEY = "your_groq_api_key_here"

# Launch KALI
streamlit run Mission_Control.py
```

### 🇮🇳 India Data & Research Context
- **Market Data**: Sourced from Yahoo Finance (NSE India suffix `.NS`).
- **Benchmark**: NIFTY 50 Index (`^NSEI`).
- **Research Examples**: Integrated context from the National Quantum Mission, ISRO's quantum experiments, and IIT Madras research nodes.

### 👩‍🏫 Student Instructions
1. **Choose a Topic**: Start with "What is a Qubit?" in the sidebar curriculum.
2. **Take the Quiz**: Complete the assessment at the end of each lesson to earn your 🟢 Progress Dot.
3. **Master the Challenge**: Achieving 3/3 on a quiz unlocks the **Bonus Challenge** for advanced learning.
4. **Optimize Assets**: Use the BEE Dashboard to see how quantum principles solve real-world investment problems in ₹.
5. **Ask Anything**: Use the chat panel at the bottom of KALI's portal to ask questions anytime.
