# Portfolio Optimization Data Sources

All data used in this platform is sourced from Indian market indices and sectoral equivalents.

## India-Based Data Sources

1. **National Stock Exchange (NSE)**: Primary source for stock return simulations and volatility metrics.
2. **NIFTY 50 Index**: Benchmarking data for portfolio optimization.
3. **Reserve Bank of India (RBI)**: Source for risk-free rate assumptions and currency exchange metrics (INR-based).
4. **Local Sectoral Analysis**: Research deliverable A1 and A2 use simulated asset universes derived from Indian Large-Cap and Mid-Cap sectors.
